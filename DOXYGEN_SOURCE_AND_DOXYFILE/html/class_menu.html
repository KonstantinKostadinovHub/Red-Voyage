◆ EnlargeButtons()

void Menu::EnlargeButtons ( coordinates  coor,
SDL_Rect &  rect,
SDL_Rect  startRect 
)

Function used to create a enlargment effect when the mouse hovers over a button.

The function accepts 3 parameters

Parameters
coordinatescoor - coordinates of the mouse
SDL_Rect&rect - the Rect we are going to modify (that's why its passed by reference)
SDL_RectstartRect - the starting Rect that we can always come back to

Defines:

Parameters
currentBWA variable that shows how much a button has grown until now on his width*‍/
currentBHA variable that shows how much a button has grown until now on his height
bonusWA variable that shows how much a button can grown on his width per update
bonusHA variable that shows how much a button can grown on his height per update
maxWidthA variable that shows how much a button is allowed to grow on his width
maxHeightA variable that shows how much a button is allowed to grow on his height

After checking if the mouse is over a button and it is within the allowed width and height we perform the enlargement formulas

If the mouse doesnt hover over a button we return it to its original size
